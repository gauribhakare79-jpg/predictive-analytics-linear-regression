# Predictive Analytics using Linear Regression

## Project Overview
This project demonstrates predictive analytics using historical sales data. A Linear Regression model is trained to predict future sales based on historical data.

## Features
- Data preprocessing
- Feature engineering
- Linear Regression model
- Model evaluation
- Data visualization

## Technologies Used
- Python
- Pandas
- NumPy
- Matplotlib
- Scikit-learn

## Workflow
1. Load dataset
2. Preprocess data
3. Feature engineering
4. Train-test split
5. Train Linear Regression model
6. Predict sales
7. Evaluate model
8. Visualize predictions

## Evaluation Metrics
- Mean Absolute Error (MAE)
- Mean Squared Error (MSE)
- Root Mean Squared Error (RMSE)
- R² Score

## Output
The project predicts future sales using historical data and visualizes the comparison between actual and predicted sales.
